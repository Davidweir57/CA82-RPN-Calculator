Template repo for CA282 week 9.

This is a npm project.  It depends upon a number of npm modules.

To install the necessary npm modules, use "npm install".

To build the static/index.html file, use:

  - either "Make -C static build"

  - or:
    cd static && make build

If you do not have asciidoc, then read the comment in static/Makefile.
